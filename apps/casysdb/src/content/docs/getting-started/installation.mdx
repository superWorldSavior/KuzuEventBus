CasysDB ships as a native library. Today the Python SDK is available via `pip`, and the TypeScript bindings are under active development.

## Python

CasysDB provides prebuilt wheels for CPython 3.8 and newer on the major platforms.

### Install from PyPI

```bash
pip install casys-db
```

### Build from source

Use this workflow if you are contributing to the engine or targeting a platform that does not yet have prebuilt wheels.

```bash
git clone https://github.com/casysai/casysdb.git
cd casysdb/sdk/python
pip install maturin
maturin develop --release
```

**Prerequisites**

- Python 3.8 or newer
- `pip`, `maturin`, and a Rust toolchain (for source builds)

### Verify the installation

```python
from casys_db import Database

db = Database(":memory:")
print("CasysDB ready:", db)
```

## TypeScript / Node.js

The TypeScript SDK is currently in development. Follow the project on GitHub for release updates, or build the native engine and drive it via the low-level FFI if you need early access.
