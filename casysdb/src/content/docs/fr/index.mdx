## Qu'est-ce que CasysDB ?

CasysDB est un moteur de base de données graphe moderne et embarqué, construit en Rust avec des bindings natifs pour Python et TypeScript. Il implémente **ISO GQL** (Graph Query Language) et fournit des transactions basées sur MVCC pour un accès concurrent fiable.

## Exemple Rapide

```python
from casys_db import Database, NodeEntity

class Person(NodeEntity):
    pass

# Ouvrir la base de données
db = Database("chemin/vers/db")
branch = db.default_branch()
session = branch.session()

# Créer des nœuds
session.execute("""
  CREATE (p:Person {name: 'Alice', age: 30})
  RETURN p
""")

# Requête avec API fluente
adults = session.Person.where(lambda p: p.age >= 18).all()
```

## Fonctionnalités Clés

- **Embarquée**: Aucun processus serveur séparé requis
- **ISO GQL**: Langage de requête graphe conforme aux standards
- **MVCC**: Isolation snapshot pour les transactions
- **Point-in-Time Recovery (PITR)**: Créez des branches depuis n'importe quel commit
- **SDKs Python & TypeScript**: Query builders fluents avec sécurité des types
- **Zéro dépendance**: Binaire unique, empreinte minimale
