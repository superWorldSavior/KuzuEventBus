CasysDB is available as a native library for **Python** and **TypeScript/Node.js**.

### Python (pip)

```bash
pip install casys-db
```

### From sources (Python)

```bash
git clone https://github.com/casysai/casysdb.git
cd casysdb/sdk/python
pip install maturin
maturin develop --release
```

#### Requirements

- Python 3.8+
- pip or poetry

### TypeScript/Node.js (coming soon)

TypeScript SDK is under development and will be available soon.

### Verify installation

```python
from casys_db import Database

print("CasysDB installed!")
db = Database(":memory:")
print(f"DB created: {db}")
```

```ts
import { Database } from 'casys-db';

console.log('CasysDB installed!');
const db = new Database(':memory:');
console.log(`DB created: ${db}`);
```
