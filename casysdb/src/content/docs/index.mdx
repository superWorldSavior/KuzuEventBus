## What is CasysDB?

CasysDB is a modern, embedded graph database engine built in Rust with native bindings for Python and TypeScript. It implements **ISO GQL** (Graph Query Language) and provides MVCC-based transactions for reliable concurrent access.

<CardGrid>
  <Card title="🚀 Fast & Embedded" icon="rocket">

  </Card>
  
  <Card title="📊 ISO GQL Support" icon="open-book">

## Quick Example

```python
from casys_db import Database, NodeEntity

class Person(NodeEntity):
    pass

# Open database
db = Database("path/to/db")
branch = db.default_branch()
session = branch.session()

# Create nodes
session.execute("""
  CREATE (p:Person {name: 'Alice', age: 30})
  RETURN p
""")

# Query with fluent API
adults = session.Person.where(lambda p: p.age >= 18).all()
```

## Key Features

- **Embedded**: No separate server process required
- **ISO GQL**: Standards-compliant graph query language
- **MVCC**: Snapshot isolation for transactions
- **Point-in-Time Recovery (PITR)**: Branch from any historical commit
- **Python & TypeScript SDKs**: Fluent query builders with type safety
- **Zero dependencies**: Single binary, minimal footprint
