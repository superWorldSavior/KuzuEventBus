# KuzuEventBus Frontend - Copilot Configuration

## Quick Reference

### Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Zustand + React Query
- React Router DOM v6
- React Hook Form + Zod

### Path Aliases

```typescript
@/*                 -> ./src/*
@/components/*      -> ./src/components/*
@/pages/*          -> ./src/pages/*
@/hooks/*          -> ./src/hooks/*
@/store/*          -> ./src/store/*
@/services/*       -> ./src/services/*
@/types/*          -> ./src/types/*
@/utils/*          -> ./src/utils/*
```

### Component Template

```typescript
interface ComponentProps {
  // Define props here
}

export function Component({}: ComponentProps) {
  return <div className="tailwind-classes">{/* Component content */}</div>;
}
```

### API Service Pattern

```typescript
// Use apiClient from @/services/api
const { data, isLoading, error } = useQuery({
  queryKey: ["resource"],
  queryFn: () => apiClient.get("/endpoint"),
});
```

### State Management

- Global state: Zustand stores in `@/store/`
- Server state: React Query
- Form state: React Hook Form with Zod validation
- Local state: useState

### Styling Guidelines

- Use Tailwind utility classes
- Use `cn()` from `@/utils` for conditional styling
- Follow shadcn/ui component patterns
- Implement responsive design

### Import Order

1. React and external libraries
2. Internal components and hooks
3. Types (with `import type`)
4. Utilities and constants
